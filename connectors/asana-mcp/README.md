# Asana MCP Connector (MVP)

This connector exposes Asana APIs to CoWork OS through MCP tools.

## Requirements

- `ASANA_ACCESS_TOKEN` (required)

## Build & Run

```bash
npm install
npm run build
npm start
```

## Tools

- `asana.health`
- `asana.list_projects`
- `asana.get_task`
- `asana.search_tasks`
- `asana.create_task`
- `asana.update_task`
