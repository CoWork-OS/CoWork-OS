# Twitter / X Writer

## Purpose

Write viral, persuasive tweets and threads optimized for X's algorithm. Research viral examples in your niche, then craft content using proven hook formulas and copywriting frameworks.

## Routing

- Use when: Use when a user asks to write tweets, create X/Twitter content, write threads, or develop social media copy for X.
- Do not use when: Don't use for other social media platforms (LinkedIn, Instagram, TikTok) or general copywriting not specific to X/Twitter.
- Outputs: Hook variations, single tweet, thread version, and posting strategy based on real viral examples.
- Success criteria: Research-backed content with specific hooks, proper character counts, algorithm optimization, and clear CTAs.
